Java环境搭建与语法入门

发表于 2021-11-25 更新于 2025-06-14

Java简介

  • 面向对象
  • 1995年由sun公司发布
  • 2010年sun公司被Oracle公司收购

JVM

  • Java virtual machine Java虚拟机
  • 是Java平台无关性实现的关键 ,即把字节码文件解释成具体平台上的机器指令执行

Java程序执行过程

  • 两个组件:

    ​ -javac -编译器,将源程序转成字节码(.class)

    ​ -java -运行编译后的字节码

JRE

  • Java runtime environment,包括jvm,Java核心类库和支持文件
  • jre=jvm+Javase标准类库

Java平台

  • Java SE 桌面程序 Java标准版
  • Java EE web程序 Java企业版
  • Java ME 移动设备 Java微型版

标识符

程序员自己定义的名称，例如类名，方法名，变量等。

  • 不能以数字开头,不能有空格
  • 区分大小写
  • 不能是Java关键字和保留字（Void是合法的,void是不合法的）

关键字

Java中的关键字是有对编译器有特殊意义的词。比如class是用来定义类的关键字，编译器遇到class就知道这是定义了一个类。

保留字

保留字是Java预留的关键字，虽然现在还没有作为关键字，但是以后的升级版本中会成为关键字，保留字包括：goto、const。

变量

计算机中临时存储在内存中的数据

  • 满足标识符命名规则
  • 驼峰法命名，age，stuName

类的命名

  • 满足Pascal法命名，HelloJava

数据类型

  • 基本数据类型 说明 字节数 默认值 min max
    byte 字节型 1 0 -2^7 2^7-1
    short 短整型 2 0 -2^15 2^15 - 1
    int 整形 4 0 -2^31 2^31 - 1
    long 长整形 8 0L -2^63 2^63 -1
    float 单精度浮点型 4 0.0f -3.40E+38 3.40E+38
    double 双精度浮点型 8 0.0d -1.79E+308 1.79E+308
    char 字符型 2 0 65535
    boolean 布尔型 1 false
    引用数据类型
    class 类
    interface 接口
    array 数组

进制表示

  • 八进制: 0开头,包括0-7的数字,如023
  • 十六进制: 以0x或0X开头,包括0-9的数字,及字母a-f,A-F,如0x1abcL(L代表长整型)

逻辑运算符

&& 运算符是短路运算符,如果第一个表达式为false，结果为false,运算符右边的表达式就不再计算了。即只要有一个操作数为false结果为false。

|运算符只要有一个为true结果为true

||运算符也是短路运算符，如果第一个表达式的值为true，结果为true，运算符右边的表达式就不再计算了。
