标识符

程序员自己定义的名称，例如类名，方法名，变量等。

  • 不能以数字开头,不能有空格
  • 区分大小写
  • 不能是Java关键字和保留字（Void是合法的,void是不合法的）

关键字

Java中的关键字是有对编译器有特殊意义的词。比如class是用来定义类的关键字，编译器遇到class就知道这是定义了一个类。

保留字

保留字是Java预留的关键字，虽然现在还没有作为关键字，但是以后的升级版本中会成为关键字，保留字包括：goto、const。

package com.kuroneko;

public class TypeExchange {
	public static void main(String[] args) {
		//char类型和int类型之间的转换
		char c= (char)65536;
		int n;
		n= c; //隐式类型转换
		c= (char)n;	//强制转换
		
		//整型和浮点型的类型转换问题
		int x= 100;
		long y= x;
		x=(int)y;
		float f= 100000000000000L;
		System.out.println("f="+ f);
		float f1= 103948583923948L;
		System.out.println("f1="+ f1);
	}

}

闰年问题

用if-else判断输入的年份是否为闰年

闰年的判断规则：能被4整除但不能被100整除的年份，或者能被400整除的年份

package com.kuroneko.operator;

import java.util.Scanner;

public class LeapYearDemo {
	public static void main(String[] args) {
		System.out.print("请输入年份:");
		Scanner sc=new Scanner(System.in);
		int year=sc.nextInt();
		//闰年的判断规则：能被4整除但不能被100整除的年份，或者能被400整除的年份
		if(((year%4==0) & (year%100!=0))|(year%400==0)) {
			System.out.print(year+"是闰年！");
		}else {
			System.out.print(year+"不是闰年！");
		}
	}

}

public class ScoreAssess {
	public static void main(String[] args) {

		 * 编写一个程序,根据考试成绩,输出相应的评定信息
		 * 成绩>=90,优
		 * 成绩[80,90),良
		 * 成绩[60,80),中
		 * 成绩<60,不及格
		 */
		System.out.println("请输入成绩:");
		Scanner sc=new Scanner(System.in);
		int score=sc.nextInt();
		if(score>=90) 
			System.out.println("优");
		else if(score>=80)
			System.out.println("良");
		else if(score>=60)
			System.out.println("中");
		else
			System.out.println("不及格");
		
		
	}
}
