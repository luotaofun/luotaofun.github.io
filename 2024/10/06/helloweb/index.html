让子元素（块元素）在父元素中垂直居中布局

<!DOCTYPE html>
<html lang="zh-CN">
    <head>
        <meta charset="utf-8">
        <title>让子元素（块元素）在父元素中垂直居中布局</title>
        <style>
            .outer{
                width: 400px;
                height: 400px;
                background-color: gray;
                /* 解决margin塌陷 */
                overflow: hidden;
            }
            .inner{
                width: 200px;
                height: 100px;
                background-color: orange;
                /*上下0 左右自动 */
                margin:0 auto;
                /*
                    上边距覆盖为（400-100）/2
                    使子元素在父元素中垂直居中
                    即：若子元素为块元素，给子元素加上margin-top，值为(父元素content-子元素盒子总高)/2
                */
                margin-top: 150px; 
                /* 水平居中 */
                text-align: center;
                /* line-height=height 时垂直居中*/
                line-height: 100px;
            }
        </style>
    </head>
    <body>
        <div class="outer">
            <div class="inner">我爱你</div>
        </div>
    </body>
    </head>
    
</html>

让子元素（行内元素、行内块元素）在父元素中垂直居中布局

<!DOCTYPE html>
<html lang="zh-CN">
    <head>
        <meta charset="utf-8">
        <title>让子元素（行内元素、行内块元素）在父元素中垂直居中布局:
            让父元素height=line-height,每个子元素都加上vertical-align: middle
            若想绝对垂直居中，可让父元素font-size设置为0
        </title>
        <style>
            .outer{
                width: 400px;
                height: 400px;
                background-color: gray;
                text-align: center;
                line-height: 400px;
                /* 消除基线"x"影响,以达到绝对的垂直居中 */
                font-size: 0px;
            }
            span{
                background-color: orange;
                vertical-align: middle;
                font-size: 20px;
            }
            img{
                width: 100px;
                vertical-align: middle;
            }
        </style>
    </head>
    <body>
        <div class="outer">
            <span>this's meX</span>
            <img src="./img/me.png">
        </div>
    </body>
    </head>
    
</html>
