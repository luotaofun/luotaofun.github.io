Java常用类库总结

Java常用类库总结

一、字符串处理类

1. String类

特点：

  • 不可变性（immutable）
  • 字符串常量池
  • 线程安全

常用方法：

// 字符串操作
String str = "Hello World";
str.length();           // 获取长度
str.substring(0, 5);    // 截取子串
str.trim();            // 去除首尾空格
str.split(",");        // 分割字符串
str.replace("l", "L"); // 替换字符

// 字符串比较
str.equals("Hello");   // 内容比较
str.equalsIgnoreCase("hello"); // 忽略大小写比较
str.startsWith("He");  // 前缀判断
str.endsWith("ld");    // 后缀判断

// 字符串查找
str.indexOf("o");      // 查找字符位置
str.contains("World"); // 包含判断

2. StringBuilder和StringBuffer

特点：

  • StringBuilder：非线程安全，性能好
  • StringBuffer：线程安全，性能较差

使用场景：

// 字符串拼接
StringBuilder sb = new StringBuilder();
sb.append("Hello");
sb.append(" ");
sb.append("World");
String result = sb.toString();

// 链式调用
StringBuilder sb = new StringBuilder()
    .append("Hello")
    .append(" ")
    .append("World");

二、日期时间类

1. LocalDate/LocalTime/LocalDateTime

特点：

  • 不可变性
  • 线程安全
  • 更直观的API

常用方法：

// 获取当前日期时间
LocalDate today = LocalDate.now();
LocalTime now = LocalTime.now();
LocalDateTime dateTime = LocalDateTime.now();

// 日期操作
LocalDate tomorrow = today.plusDays(1);
LocalDate lastMonth = today.minusMonths(1);

// 日期格式化
DateTimeFormatter formatter = DateTimeFormatter.ofPattern("yyyy-MM-dd");
String formatted = today.format(formatter);

// 日期比较
boolean isAfter = today.isAfter(tomorrow);
boolean isBefore = today.isBefore(tomorrow);

2. DateTimeFormatter

特点：

  • 线程安全
  • 预定义格式
  • 自定义格式

使用示例：

// 预定义格式
DateTimeFormatter formatter = DateTimeFormatter.ISO_LOCAL_DATE;
String formatted = dateTime.format(formatter);

// 自定义格式
DateTimeFormatter customFormatter = DateTimeFormatter.ofPattern("yyyy年MM月dd日 HH:mm:ss");
String customFormatted = dateTime.format(customFormatter);

三、数值处理类

1. Math类

常用方法：

// 数学运算
Math.abs(-10);     // 绝对值
Math.max(10, 20);  // 最大值
Math.min(10, 20);  // 最小值
Math.round(3.14);  // 四舍五入
Math.floor(3.14);  // 向下取整
Math.ceil(3.14);   // 向上取整

// 随机数
Math.random();     // 0-1之间的随机数

2. Random类

特点：

  • 伪随机数生成器
  • 可设置种子
  • 多种随机数类型

使用示例：

Random random = new Random();

// 生成随机数
int randomInt = random.nextInt(100);    // 0-99
double randomDouble = random.nextDouble(); // 0-1
boolean randomBoolean = random.nextBoolean();

// 设置种子
Random seededRandom = new Random(42L);

四、集合类

1. Collections工具类

常用方法：

List<String> list = new ArrayList<>();

// 排序
Collections.sort(list);
Collections.sort(list, Collections.reverseOrder());

// 查找
int index = Collections.binarySearch(list, "key");

// 填充
Collections.fill(list, "default");

// 同步包装
List<String> syncList = Collections.synchronizedList(list);

2. Arrays工具类

方法 说明
equals(arr1, arr2) 比较两个数组是否相等
sort(arr) 对数组元素进行从小到大排序
toString(arr) 将数组转换成字符串
fill(arr, value) 将 value 值赋给数组中的每一个元素
fill(arr, start, end, value) 将 value 值赋给数组中指定范围的元素
copyOf(arr, length) 把数组复制成指定长度的新数组
copyOfRange(arr, start, end) 将数组的指定范围复制到新数组中
binarySearch(arr, value) 查找数组元素，返回下标

常用方法：

int[] array = {1, 2, 3, 4, 5};

// 排序
Arrays.sort(array);

// 二分查找
int index = Arrays.binarySearch(array, 3);

// 填充
Arrays.fill(array, 0);

// 复制
int[] copy = Arrays.copyOf(array, array.length);

五、IO类

1. File类

常用方法：

File file = new File("test.txt");

// 文件操作
file.exists();           // 是否存在
file.isFile();          // 是否是文件
file.isDirectory();     // 是否是目录
file.createNewFile();   // 创建文件
file.delete();          // 删除文件
file.mkdir();           // 创建目录

// 文件信息
file.getName();         // 文件名
file.getPath();         // 路径
file.length();          // 文件大小
file.lastModified();    // 最后修改时间

2. 流操作

字节流：

// 文件复制
try (FileInputStream fis = new FileInputStream("source.txt");
     FileOutputStream fos = new FileOutputStream("target.txt")) {
    byte[] buffer = new byte[1024];
    int length;
    while ((length = fis.read(buffer)) > 0) {
        fos.write(buffer, 0, length);
    }
}

字符流：

// 文本文件读写
try (BufferedReader reader = new BufferedReader(new FileReader("input.txt"));
     BufferedWriter writer = new BufferedWriter(new FileWriter("output.txt"))) {
    String line;
    while ((line = reader.readLine()) != null) {
        writer.write(line);
        writer.newLine();
    }
}

六、反射类

1. Class类

常用方法：

// 获取Class对象
Class<?> clazz = String.class;
Class<?> clazz2 = Class.forName("java.lang.String");

// 创建实例
Object obj = clazz.newInstance();

// 获取方法
Method method = clazz.getMethod("length");

// 获取字段
Field field = clazz.getDeclaredField("value");

// 获取构造器
Constructor<?> constructor = clazz.getConstructor();

2. 反射应用

动态代理：

// 创建代理
Proxy.newProxyInstance(
    target.getClass().getClassLoader(),
    target.getClass().getInterfaces(),
    (proxy, method, args) -> {
        // 前置处理
        Object result = method.invoke(target, args);
        // 后置处理
        return result;
    }
);

七、异常处理类

1. 异常体系

try {
    // 可能抛出异常的代码
} catch (IOException e) {
    // 处理IO异常
} catch (Exception e) {
    // 处理其他异常
} finally {
    // 清理资源
}

2. 自定义异常

public class CustomException extends Exception {
    public CustomException(String message) {
        super(message);
    }
    
    public CustomException(String message, Throwable cause) {
        super(message, cause);
    }
}

八、线程类

1. Thread类

创建线程：

// 方式1：继承Thread
class MyThread extends Thread {
    @Override
    public void run() {
        // 线程执行代码
    }
}

// 方式2：实现Runnable
class MyRunnable implements Runnable {
    @Override
    public void run() {
        // 线程执行代码
    }
}

// 方式3：Lambda表达式
Thread thread = new Thread(() -> {
    // 线程执行代码
});

2. 线程池

// 创建线程池
ExecutorService executor = Executors.newFixedThreadPool(5);

// 提交任务
executor.submit(() -> {
    // 任务代码
});

// 关闭线程池
executor.shutdown();

九、网络类

1. URL类

URL url = new URL("https://www.example.com");

// 获取连接
URLConnection conn = url.openConnection();

// 读取数据
try (BufferedReader reader = new BufferedReader(
        new InputStreamReader(conn.getInputStream()))) {
    String line;
    while ((line = reader.readLine()) != null) {
        System.out.println(line);
    }
}

2. Socket类

// 服务器端
ServerSocket server = new ServerSocket(8080);
Socket socket = server.accept();

// 客户端
Socket client = new Socket("localhost", 8080);

十、实用工具类

1. Objects类

// 空值检查
Objects.isNull(obj);
Objects.nonNull(obj);
Objects.requireNonNull(obj);

// 比较
Objects.equals(obj1, obj2);
Objects.deepEquals(obj1, obj2);

2. Optional类

// 创建Optional
Optional<String> optional = Optional.of("value");
Optional<String> empty = Optional.empty();

// 使用Optional
optional.ifPresent(System.out::println);
String value = optional.orElse("default");

1. 性能优化

  • 使用StringBuilder代替String拼接
  • 使用try-with-resources自动关闭资源
  • 合理使用线程池
  • 使用批量操作代替循环操作

2. 代码规范

  • 使用常量类管理常量
  • 使用枚举代替魔法数字
  • 使用Optional处理空值
  • 使用try-with-resources处理资源

3. 异常处理

  • 使用具体的异常类型
  • 合理使用finally块
  • 记录异常日志
  • 自定义业务异常

4. 线程安全

  • 使用线程安全的集合类
  • 使用synchronized关键字
  • 使用volatile关键字
  • 使用原子类
