str.startsWith("He");  // еүҚзјҖеҲӨж–ӯ
str.endsWith("ld");    // еҗҺзјҖеҲӨж–ӯ

// еӯ—з¬ҰдёІжҹҘжүҫ

2. Arraysе·Ҙе…·зұ»

ж–№жі• иҜҙжҳҺ

fill(arr, start, end, value) е°Ҷ value еҖјиөӢз»ҷж•°з»„дёӯжҢҮе®ҡиҢғеӣҙзҡ„е…ғзҙ

copyOfRange(arr, start, end) е°Ҷж•°з»„зҡ„жҢҮе®ҡиҢғеӣҙеӨҚеҲ¶еҲ°ж–°ж•°з»„дёӯ
