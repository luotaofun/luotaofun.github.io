Java多线程与IO学习笔记

1. 快递站线程池模型（ThreadPool工作原理）

// 快递分拣中心类比线程池
ExecutorService pool = Executors.newFixedThreadPool(5);

// 包裹分拣任务
Runnable packageTask = () -> {
    System.out.println(Thread.currentThread().getName() + "正在分拣包裹");
};

// 提交10个包裹任务
IntStream.rangeClosed(1,10).forEach(i -> pool.execute(packageTask));

2. 高速公路同步锁机制

// 收费站闸机同步示例
ReentrantLock tollGateLock = new ReentrantLock();

void vehiclePass() {
    tollGateLock.lock();
    try {
        // 每次只允许一辆车通过
        System.out.println(Thread.currentThread().getName() + "正在缴费通过");
    } finally {
        tollGateLock.unlock();
    }
}

6. 生产者-消费者对话式实现

BlockingQueue<String> conveyor = new ArrayBlockingQueue<>(10);

// 生产线程
new Thread(() -> {
    while(true) {
        String product = "包裹"+System.currentTimeMillis();
        conveyor.put(product);
        System.out.println("生产：" + product);
    }
}).start();

// 消费线程
new Thread(() -> {
    while(true) {
        String product = conveyor.take();
        System.out.println("消费：" + product);
        Thread.sleep(1000);
    }
}).start();

Java多线程与IO学习笔记

一、多线程

1. 什么是多线程

想象一下，你就像一个超级厨师，在厨房里同时做几道菜。单线程就好比你一次只能专注做一道菜，做完一道再做下一道；而多线程呢，就像是你可以同时处理好几道菜，这边炒着菜，那边煮着汤，还能同时切着水果。这样效率是不是就大大提高啦？在计算机里，多线程就是让程序可以同时执行多个任务，充分利用CPU的资源。

2. 多线程的使用方法

在Java里，要创建多线程有两种常见的方法。一种是继承Thread类，就好像你开了一家新的餐厅分店，这个分店有自己独特的经营方式（也就是线程的执行逻辑）。另一种是实现Runnable接口，这就好比你有一个通用的经营方案，很多餐厅都可以按照这个方案来经营（多个线程可以共享同一个任务逻辑）。

继承Thread类示例代码

class MyThread extends Thread {
    public void run() {
        System.out.println("这是一个继承Thread类的线程在执行任务");
    }
}

public class Main {
    public static void main(String[] args) {
        MyThread thread = new MyThread();
        thread.start();
    }
}

实现Runnable接口示例代码

class MyRunnable implements Runnable {
    public void run() {
        System.out.println("这是一个实现Runnable接口的线程在执行任务");
    }
}

public class Main {
    public static void main(String[] args) {
        MyRunnable myRunnable = new MyRunnable();
        Thread thread = new Thread(myRunnable);
        thread.start();
    }
}

3. 多线程的注意事项

多线程虽然能提高效率，但也可能会带来一些问题，就像餐厅同时接待很多客人，可能会出现服务混乱的情况。在多线程里，最常见的问题就是线程安全问题，比如多个线程同时访问和修改同一个数据，就可能会导致数据不一致。所以，我们需要使用一些同步机制来保证线程安全，就像餐厅要制定一些规则来保证服务的有序进行。

二、IO（输入输出）

1. 什么是IO

IO就像是你和外界交流的通道。想象一下，你要从外面的世界获取信息，就像你要从超市里买东西回家，这就是输入；你要把自己的想法告诉别人，就像你要把家里的东西卖给别人，这就是输出。在计算机里，IO就是程序和外部设备（如文件、网络等）之间进行数据交换的过程。

2. IO的分类

Java里的IO可以分为字节流和字符流。字节流就像是一个个小的水滴，它可以处理任何类型的数据，包括图片、音频等；字符流就像是一个个文字，它主要用于处理文本数据。

3. IO的使用方法

文件读取示例（字节流）

import java.io.FileInputStream;
import java.io.IOException;

public class FileReadExample {
    public static void main(String[] args) {
        try (FileInputStream fis = new FileInputStream("test.txt")) {
            int data;
            while ((data = fis.read()) != -1) {
                System.out.print((char) data);
            }
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

文件写入示例（字节流）

import java.io.FileOutputStream;
import java.io.IOException;

public class FileWriteExample {
    public static void main(String[] args) {
        try (FileOutputStream fos = new FileOutputStream("test.txt")) {
            String content = "这是要写入文件的内容";
            byte[] bytes = content.getBytes();
            fos.write(bytes);
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}
